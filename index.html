How to use

  1. Add MIDI files and select your source and destination libraries.
  2. Any unmapped articulations will be shown per-file before you convert.
  3. Downloaded output includes a folder of individual MIDI files for each unmapped articulation, named with the source library's articulation and note number. Import these into your DAW alongside the main converted file to manually assign them to the articulation of your choice.
